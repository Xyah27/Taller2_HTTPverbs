GET   -- Su usa para obtener los datos.
POST   - Se usa para crear un nuevo recursso.
PUT   -- Se utiliza pra actualizar completamente un recurso existente en la base de datos.
DELETE - Se utiliza para eliminar un recurso del servidor
HEAD  -- Se utiliza para obtener los encabezados HTTP de la respuesta sin el cuerpo.
OPTION - Se usa para solicitar al servidor las operaciones HTTP permitidas en un recurso.
PATCH -- Actualiza parcialmente un recurso ya existente.
